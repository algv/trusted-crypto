# trusted-crypto
Trusted crypto library [![Build Status](https://travis-ci.org/algv/trusted-crypto.svg?branch=master&style=flat)](https://travis-ci.org/algv/trusted-crypto)

## Install dependencies

```
> npm install -g typescript
> npm install -g tsd
> npm install -g mocha
```

## Build

```
> npm install
```

## Run test

```
> npm test
```
